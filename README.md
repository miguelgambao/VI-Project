# VI-Project
Plane crash visualizer
Miguel Gambão & António Simões

Contains multiple visualizers of data from an enhanced version of the Airplane Crashes 1908-2009 data-set (https://www.kaggle.com/datasets/saurograndi/airplane-crashes-since-1908) and using mainly D3 graph.

All of the visualizers use a set of filters that affect them all.

Right now there are map and graph views.

Map - See location of crashes, hover for tooltips for info on the crash and toggle routes

Graphs - Switch between diferent types of graphs and their settings.
