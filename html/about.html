About This Project

This visualization explores historical plane crash data (1940-2009), providing interactive tools to analyze patterns across time, geography, weather conditions, and impact metrics.

Map View: Visualize crash locations on a world map with route visualization. Filter crashes by year range, number of passengers aboard, fatality percentage, and weather conditions. Hover over crash locations to see detailed information.

Graphs: Analyze crash trends over time with flexible data visualization. Toggle between displaying crashes only, fatalities only, or both simultaneously. Compare datasets using dual Y-axis scales for accurate representation. View data as time-series line charts or aggregated by decade in pie charts.

Scatter Matrix: Explore correlations between weather attributes (temperature, precipitation, wind, cloud cover, pressure) and crash patterns. Points are color-coded by weather condition (Severe, Rain, Snow, Fog, Windy, Cloudy, Clear, Other).

All views share synchronized filters for consistent data exploration across the entire dataset.
